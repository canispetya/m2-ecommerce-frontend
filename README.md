# M2 — E-commerce Frontend

## Descripción
Interfaz web de una tienda e-commerce con navegación, listado de productos, detalle y footer.

## Tecnologías
- HTML5, CSS3, JavaScript (vanilla)
- (Opcional) Bootstrap por CDN

## Ejecutar el proyecto
- Opción 1: abrir `index.html` en el navegador.
- Opción 2 (recomendada): usar una extensión de servidor local (Live Server en VS Code).

## Funcionalidades
- Navbar para navegar por la página
- Listado de productos en tarjetas con botón de compra
- Página de detalle de producto
- Footer con información de la tienda

## Comandos básicos de Git

### git --version

**Definición breve:**
Muestra la versión de Git instalada en el sistema, útil para verificar que Git está correctamente instalado.

**Ejemplo de uso:**

```bash
git --version
```

---

### git init

**Definición breve:**
Inicializa un nuevo repositorio Git en la carpeta actual, creando el directorio oculto `.git`.

**Ejemplo de uso:**

```bash
git init
```

---

### git add

**Definición breve:**
Agrega archivos o cambios al área de preparación (staging), dejándolos listos para ser confirmados.

**Ejemplo de uso:**

```bash
git add .
```

_(Agrega todos los archivos modificados y nuevos)_

---

### git commit

**Definición breve:**
Registra los cambios del área de preparación en el historial del repositorio junto con un mensaje descriptivo.

**Ejemplo de uso:**

```bash
git commit -m "Primer commit"
```

---

### git status

**Definición breve:**
Muestra el estado actual del repositorio, indicando archivos modificados, preparados o sin seguimiento.

**Ejemplo de uso:**

```bash
git status
```

---

## Flujo típico de trabajo en Git

```bash
git add .
git commit -m "mensaje"
git status
```
